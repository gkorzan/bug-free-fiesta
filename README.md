**Description**
---
A rogue-like game written in rust using tcod library

## Startup
It is a **Rust** project you will need *rustc* and *cargo*
    
    $ git clone git@github.com:gkorzan/bug-free-fiesta.git
    $ cd bug-free-fiesta
    $ cargo run
